Using the cli tool
==================

Invoke the tool with: ::

        bw2-browser

The tool is built around "commands". You type a command, and the tool will execute it.
commands are mostly single letters (with a few 3 letter ones).

Example
-------

Calculate all the scores for a method, for an activity.

1. invoke the tool.

   ``bw2-browser``

2. select the right project.

   *From the list, type the number of the project and hit enter.*

3. select the right database.

   *From the list, type the number of the database and hit enter.*

4. select the right activity.

   *Use the* ``s`` *command to search for an activity. type the number of the activity and hit enter.*

5. select the method.

   *Use the* ``lm`` *command to list the methods. type the number of the method and hit enter.*

6. Calculate the LCIA score of the selected activity and method(s).

   *Use the* ``G`` *command.*
